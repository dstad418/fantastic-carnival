Projects

  • x86: This is project in xv6 architecture where I recreated a lottery scheduler using the C-language based on the size of executing instruction.
    This wasn't my only foray into xv6 either, as there are a few projects on my github involving replacements for the malloc family of calls, bitarray multi-threading and matrix-multiplication multi-threading.
    I encourage you to check them out!

  • Mars Turbine Program: This program simulates gathering and parsing high-volumes of data, while keeping it organized in three-parallel linked-lists; it also allows the user to query and process certain subsets of data.
    Additional projects following this theme involve a simulation that harvests and culls crops based on demand, one that uses hashmapping to sort and recall large maounts of data, and a builder that uses stacks and queues to process a list of commands.
    Feel free to look around!

  • ChocAn: This is a C++ program that was developed simulating an Agile development process, and was built over the term using incremental development. I acted as the lead developer and architect for the software design.
    In addition to the program itself, we also provided a requirements document, a test plan and a design for the system!
